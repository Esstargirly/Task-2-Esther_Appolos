# Expense Tracker

A simple command-line expense tracker built with Python. Enter your expenses one by one and the program keeps a running total, then displays a full summary at the end.

## Features

- Add multiple expense amounts interactively
- Displays a running total after each entry
- Shows a full expense summary at the end
- Handles invalid inputs gracefully

## Requirements

- Python 3.x

## How to Run

```bash
python expense_tracker.py
```

## Key Concept

This project demonstrates the use of an **accumulator pattern** — a fundamental concept in backend data processing:

```python
total = total + new_expense
```
